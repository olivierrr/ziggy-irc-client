ziggy-IRC-client
===================

##install
`npm install -g zirc`

##launch
`zirc`

![alt tag](http://i.imgur.com/N9LOqjX.png)

## available commands

* `/help`
* `/me`
* `/msg`
* `/nick`
* `/join`
* `/topic`
* `/whois`
* `/invite`
* `/notice`
